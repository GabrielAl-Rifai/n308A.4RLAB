## About The Project


This project utilizes the Dog API to fetch and display images of various dog breeds. It provides an interactive interface for users to select a breed from a dropdown menu and view a carousel of images for that breed.

Here's why it's awesome:
* Users can easily browse through a wide range of dog breeds and view images of their favorite breeds.
* It's a simple and intuitive way to explore the world of dogs!
* The project uses modern web development technologies to create an engaging user experience.

<p align="right">(<a href="#readme-top">back to top</a>)</p>

### Built With

* [Axios](https://axios-http.com)
* [Bootstrap](https://getbootstrap.com)
* [Dog API](https://thedogapi.com)
